# signd-reactnative-sdk

SignD SDK for React Native

## Installation

```sh
npm install signd-reactnative-sdk
```

## Usage

```js
import { SigndReactnativeSdkView } from "signd-reactnative-sdk";

// ...

<SigndReactnativeSdkView color="tomato" />
```
